Tip: Hover any button or link to see the cursor lock and grow. Click anywhere to see a ripple. On mobile the custom cursor is removed for usability and you'll see the touch indicator in the corner.

JS Techniques Used

  • Pointer events: pointermove, pointerdown, pointerup
  • requestAnimationFrame for smooth cursor follow & lerp motion
  • Event delegation for hover/lock behavior
  • DOM manipulation & class toggling for interaction states
  • Touch detection & accessible fallbacks
  • Dynamic element creation for click ripple and trailing blobs

Responsive behavior

  • Media queries to hide custom cursor on small screens
  • Touch indicator for tap-based devices
